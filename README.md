Calculator App

A simple and responsive calculator built with HTML, CSS, and JavaScript. This project performs basic arithmetic operations and demonstrates core JavaScript concepts such as DOM manipulation, event handling, and application logic.

🚀 Features

- Perform basic arithmetic operations
  - Addition (+)
  - Subtraction (-)
  - Multiplication (×)
  - Division (÷)
- Clear all input
- Delete the last entered character
- Responsive design for desktop and mobile devices
- Clean and intuitive user interface

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
